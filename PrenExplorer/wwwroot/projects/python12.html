Урок 12. Матрицы

Матрица в Python – это двумерный массив, поэтому задание матриц того или иного вида предполагает создание соответствующего массива. Для работы с массивами в Python используется тип данных список (англ. list). 

Пример создания и заполнения матрицы:

# импортируем библиотеку random

import random

# задаем размерность матрицы 4х6

n = 4

m = 6

# создаем список

matrix = []

# формируем исходную матрицу

# перебираем строки матрицы

for i in range(n):

# создаем список в каждой строке

  line = []

# перебираем столбцы матрицы

  for j in range(m):

# добавляем в список line случайное число от -5 до 5

   line.append(random.randint(-5,6))

# добавляем список line в список matrix, таким образом и получается матрицы

  matrix.append(line)

# вывод исходной матрицы

for line in matrix:

  print(line)

Результат:

[6, 0, -4, -4, 2, 5]

[-4, 3, -4, -4, 0, -5]

[-5, -4, 1, 1, 3, -3]

[3, -2, -5, -5, 5, 3]

Далее работа с матрицами осуществляется так же как и со списками – с помощью циклов.

Задание:

1. Дана матрица 6x6, заменить все отрицательные элементы на 0.

2. Дана матрица 5x5, создать список четных элементов массива и список нечетных.

3. Заменить элементы, находящиеся на четных индексах на 0, на нечетных на 1

4. Найти сумму элементов главной и побочной диагонали матрицы

5. В матрице заменить последний элемент каждой строки на сумму предыдущих элементов той же строки.

6. Дана матрица 6х6. В каждой строке найти сумму всех элементов, если эта сумма отрицательная, заменить все отрицательные элементы на 0. Иначе заменить все положительные элементы на сумму.

7. Дана матрица 4х4. Найти максимальный элемент в каждой строке и его индекс. Если этот элемент больше 7, заменить все элементы, стоящие на четных индексах этим максимумом. Иначе заменить все элементы, стоящие на нечетных индексах нулями.

8. Дана матрица 5х6. Найти максимальный и минимальный элемент во всей матрице и вывести их индексы. Заменить все элементы четных строк на максимальный элемент. А все элементы нечетных строк на минимальный.

9. Дана матрица 5х5. Найти максимальный элемент главной диагонали и минимальный элемент побочной диагонали и поменять их местами.

10. Дана матрица целых чисел. Вводится число. Выяснить, какие строки и столбцы матрицы с

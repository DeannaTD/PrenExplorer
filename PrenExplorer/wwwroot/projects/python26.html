Урок 25. Модуль datetime

В модуле datetime размещены классы для работы с датами и временем.

datetime.date(year, month, day)

Принимает целочисленные год, месяц и день. Возвращает объект стандартной даты -- неизменяемый объект. Атрибуты: year, month, day.

datetime.time(hour=0, minute=0, second=0, microsecond=0, tzinfo=None)

Принимает целочисленные, возвращает объект -- стандартное время. Не зависит от даты. Атрибуты: hour, minute, second, microsecond, tzinfo

datetime.datetime(year, month, day, hour=0, minute=0, second=0, microsecond=0, tzinfo=None)

Принимает целочисленные. При этом, часы, минуты, секунды и миллисекунды по умолчанию равны нулю. Возвращает объект -- комбинацию даты и времени.

datetime.timedelta

Возвращает объект, который содержит информацию о разнице между двумя моментами времени, с точностью до микросекунд.

import datetime

d1 = datetime.datetime.today()

d2 = datetime.datetime(d1.year, d1.month, d1.day-1, d1.hour, d1.minute, d1.second, d1.microsecond)

r = d1 - d2

td = datetime.timedelta(days=1)

are_equal = r == td

print(td)

print(r)

print("Are equal:", are_equal)

Атрибуты экземпляра:

Атрибут - Значение

days - Между -999999999 и 999999999 включительно

seconds - Между 0 и 86399 включительно

microseconds - Между 0 и 999999 включительно

Поддерживаемые операции:

Операция - Результат

t1 = t2 + t3 - Сумма t2 и t3. В результате мы получим t1-t2 == t3 и t1-t3 == t2

t1 = t2 - t3 - Разница между t2 и t3. В результате мы получим t1 == t2 - t3 and t2 == t1 + t3 

t1 = t2 * i или t1 = i * t2 - Дельта умноженное на целочисленное. В результате мы получим t1 // i == t2 есть истина, если i!= 0. В общем, t1 * i == t1 * (i-1) + t1 есть истина

t1 = t2 * f или t1 = f * t2 - Дельта умноженное на вещественное число. Результат округляется до ближайшего значения timedelta.resolution.

f = t2 / t3 - Деление t2 на t3. Возвращает вещественное число.

t1 = t2 / f или t1 = t2 / i - Дельта, деленное на вещественное или на целочисленное.

t1 = t2 // i или t1 = t2 // t3 - Деление с округлением вниз. Возвращает целочисленное, выкинув остаток (если таковой имеется).

t1 = t2 % t3 - Деление по модулю. Возвращает объект timedelta

q, r = divmod(t1, t2) - Деление по divmod(). При этом, возвращается 2 результата, из которых первый -- целочисленное, второй -- объект timedelta.

+t1 - Возвращает объект timedelta с тем же значением.

-t1 - Эквивалент следующего: timedelta(-t1.days, -t1.seconds, -t1.microseconds), и этого: t1* -1.

abs(t) - Эквивалент: +t когда t.days >= 0, и этого: -t когда t.days < 0.

str(t) - Возвращает строковое значение в форме [D day[s], ][H]H:MM:SS[.UUUUUU], где D является отрицательным числом если t отрицательное.

repr(t) - Возвращает строковое в форме datetime.timedelta(D[, S[, U]]), где D является негативным числом если t отрицательное.

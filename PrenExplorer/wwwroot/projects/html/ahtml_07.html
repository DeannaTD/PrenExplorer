Работа с таблицами

Сейчас мы с вами научимся создавать таблицы с помощью HTML

Думаю, вы представляете, что такое таблица в обычной жизни - это набор строк и столбцов, на пересечении которых находятся ячейки.

В HTML таблицы создаются по похожему принципу. Там тоже есть столбцы и строки с ячейками, однако HTML код таблиц при первом взгляде может показаться непривычным: таблицы создаются по рядам - сначала первый ряд, потом второй и так далее.

Теги которые испольуются при создании таблицы:

  • <table> </table> - является главным при создании таблицы
  • <tr></tr> - создает строку в таблице, которая потом разбивается на ячейки
  • <th></th> - ячейка-заголовок, текст внутри этого тега будет жирным и распологается по центру
  • <td></td> - обычная ячейка

Атрибуты для тега table:

  • border – задает ширину границ для таблицы. Значение по умолчанию - 0

  • cellspacing – управляет расстоянием между ячейками

  • width – позволяет изменять ширину таблицы

  • height – позволяет изменять высоту таблицы

Атрибут для всех тегов таблицы:

  • align – в тегах tr, th, td выравнивает текст. Если атрибут вставить в тег table, то будет перемещаться вся таблица.

    Синтаксис: У данного атрибута три значения left – вырвнивает текст по левому краю, center – выравнивает текст по центру, right – выравнивает текст по правому краю

  • bgcolor – позволяет перекрашивать задний фон в тегах таблицы

    Синтаксис: bgcolor = "red" - можем задавать английским названием цвета, или bgcolor = "#FF00A1" - в hex-формате

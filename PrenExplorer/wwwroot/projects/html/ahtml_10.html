Семантика и HTML5 теги

Семантика в прямом значении – это изучение значений слов и фраз на языке.

Т.е. семантические элементы = элементы со смыслом.

Многие элементы содержат указательные элементы, например: <div id = "nav"> <div class = "main"> <div id = "footer"> для обозначения навигации, верхнего и нижнего колонтитулов.

HTML5 предлагает новые семантические элементы для определения различных частей веб-страницы:

<article></article> Тег служащий для обозначения статьи на сайте
<aside></aside> Нужен для отображения боковых колонок на сайте
<footer></footer> Оперделяет нижний колонтикул ("подвал") страницы
<header></header> Задает заголовок ("шапку") для документа или раздела, где обычно распалагается логотип и навигация
<main></main> Указывает основное содержимое страницы
<nav></nav> Определяет навигационные ссылки на странице
<section></section> Определяет раздел на странице, обычно вставляется внутри тега main
<time></time> Определяет дату и время

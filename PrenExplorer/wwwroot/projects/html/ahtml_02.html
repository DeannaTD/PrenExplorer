Знакомство со структурой WEB страницы

Каждый html-файл имеет одинаковую структуру, которая выглядит следующим образом:

  1. <!DOCTYPE html> - каждый HTML-документ должен начинаться с объявления типа документа . Тип документа нужен, чтобы браузер мог определить версию HTML и правильно отобразить страницу, актуальная версия HTML - html5
  2. <html> - определяет начало и конец HTML файла. Все теги пишутся внутри этого тега
  3. <head> - предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными. Содержимое этого тега не отображается напрямую
  4. <title> - определяет заголовок веб-страницы, это один из важных элементов предназначенный для решения множества задач. Текст заголовка отображается в левом верхнем углу окна браузера
  5. <meta charset="utf-8"> - задает кодировку страницы, т.е для того чтобы браузер корректно отображал буквы на вашем сайте для разных языков
  6. <body> - предназначен для хранения содержания веб-страницы (контента), отображаемого в окне браузера

Формы

Основные элементы

Форма – инструмент, предназначенный для обмена данными между пользователем и сервером.

  • <form>
  • <label>Ключевое слово</label>
  • <input type="значение" name="имя" placeholder="подсказка">
  • </form>

Тег <form> устанавливает форму на веб-странице.

Перед отправкой данных браузер подготавливает информацию в виде пары "имя=значение", где имя определяется атрибутом name тега <input> или другим тегом, допустимым в форме, а значение введено пользователем или установлено в поле формы по умолчанию.

Значение вводится в зависимости от того, какая форма требуется:

  • type = "text" – текстовое поле
  • type = "radio" – поле с переключателями
  • type = "checkbox" – поле с флажками
  • type = "password" - поле пароля
  • type = "submit" – кнопка отправки
  • type = "reset" - кнопка для сброса данных формы
  • type = "number" – цифровое поле
  • type = "email" – email — поле
  • type = "file" - поле которое позволяет выбирать файл с компьютера и загрузить его в форму
  • type = "date" - поле которое позволяет вставлять дату в нашу форму

Например, следует реализовать форму:

  • <form>
  • <label>Укажите имя</label>
  • <input type="text" name="name">
  • <label>Введите пароль</label>
  • <input type="password" name="pass">
  • </form>

Нестандартные формы

Select – выпадающее меню.

Если количество вариантов для выбора занимает слишком много места, вы можете воспользоваться выпадающими меню <select>.

  • <select>
  • <option>Значение1</option>
  • <option>Значение2</option>
  • </select>

Textarea – многострочный текст. Может быть изменён размер поля.

  • <textarea>
  • …
  • </textarea>

Fieldset - предназначен для группировки элементов формы. Это облегчает работу с формами, содержащими большое количество данных. Например, один блок может быть предназначен для ввода текстовой информации, а другой — для флажков.

  • <form>
    • <fieldset>
    • <legend> Ключевое слово</legend>
    • <input ……>
    • </fieldset>
  • </form>

<button> - создает кнопку на вашей веб-странице у него есть также атрибут type. Кнопка также отправляет данные на сервер из формы. По своей работе схож с input у которого тип submit и reset. Отличие в том, что кнопку лучше поддается стилизации с помощью css.

  • <form>
  • <button type="submit">текст кнопки</button>
  • <button type="reset">текст кнопки</button>
  • </form>

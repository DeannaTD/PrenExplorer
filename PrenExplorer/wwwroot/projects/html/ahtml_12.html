Аудио и видео

HTML5 аудио

Ранее для добавления аудио на HTML-страницу было необходимо пользоваться специальными «расширениями», которые давали возможность встраивать на страницу аудио-плеер.

Начиная со стандарта HTML5, данная возможность есть по умолчанию — и реализуется с помощью тега audio.

Данный тег является парным и в самом простом случае выглядит так:

<audio src="путь к аудио-файлу" controls></audio>

Атрибут controls добавляет элементы интерфейса для плеера. Каждый браузер реализует их по-своему, и единого стандарта на этот счёт нет.

При добавлении аудио на HTML-страницу важно помнить, что не все браузеры поддерживают одни и те же форматы аудио. Специально для этого существует вложенный тег <source>, в котором указываются пути на другие форматы аудио файлов. В таком случае браузер выберет тот, который поддерживается в настоящий момент.

При такой разметке указывать src у тега <audio> не обязательно.

  • <audio controls>
  • <source src="https://example.com/audio.mp3">
  • <source src="https://example.com/audio.ogg">
  • </audio>

Задание: Добавьте аудио на страницу. Используйте аудио-файлы из интернета

Видео

Для встраивания видео контента в документ используйте элемент HTML <video>. Видео элемент может содержать один или несколько источников видео. Чтобы указать источник видео, необходимо использовать атрибут src или элемент <source> - браузер сам определит наиболее подходящий источник

Синтаксис:

  • <video>
  • <source src="адрес">
  • </video>

Атрибуты:

  • src - указывает путь к воспроизводимому видеоролику. Для этой же цели можно использовать элемент <source>. В качестве значения принимается абсолютный или относительный путь к файлу.

    Синтаксис:

    • <source src="адрес">

  • autoplay - при наличии этого атрибута видео начинает воспроизводиться автоматически после загрузки страницы. Отменяет действие preload.

    Синтаксис:

    • <video autoplay="autoplay">

  • height - задаёт высоту области для воспроизведения видеоролика. Само видео меняет свои размеры в большую или меньшую сторону, чтобы вписаться в заданные рамки, но его пропорции при этом остаются прежними. В качестве значения принимает любое целое положительное число в пикселях или процентах.

    Синтексис:

    • <video height="значение">

  • width - задаёт ширину области для воспроизведения видеоролика. Само видео меняет свои размеры в большую или меньшую сторону, чтобы вписаться в указанную ширину, но его пропорции при этом не искажаются. В качестве значения принимает любое целое положительное число в пикселях или процентах.

    Синтаксис:

    • <video width="значение">

    Значение по умолчанию: исходная ширина берётся из параметров видео. Если это значение не доступно, тогда ширина принимается равной ширине картинки, заданной атрибутом poster. Если этот атрибут не установлен, ширина устанавливается в 300 пикселей.

  • loop - зацикливает воспроизведение видео, оно повторяется каждый раз с начала после завершения. По умолчанию атрибут выключен.

    Синтаксис:

    • <video loop="loop">
    • <video loop>

  • poster - указывает адрес картинки, которая будет отображаться, пока видео не доступно или не вопроизводится. В качестве значения принимается полный или относительный путь к графическому файлу. Само изображение может быть в любом доступном формате: GIF, PNG, JPEG. Если атрибут не указан, браузер постарается отобразить первый кадр видео.

    Синтаксис:

    • <video poster="адрес">

Пример:

  • <video width="320" height="240" autoplay>
  • <source src="movie.mp4" type="video/mp4">
  • </video>

Пример нескольких источников:

  • <video width="320" height="240" autoplay>
  • <source src="movie.mp4" type="video/mp4">
  • <source src="movie.ogg" type="video/ogg">
  • </video>

Задание: Добавьте видео в формате mp4 на страницу

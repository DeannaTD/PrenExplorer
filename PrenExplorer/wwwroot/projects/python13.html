Урок 13. Словари.

Словарь (dict) представляет собой структуру данных.

Данные в словаре хранятся в формате ключ – значение.

Создание пустого словаря

# создаем пустой словарь первым способом

d1 = dict()

# выводим тип переменной d1

print(type(d1))

# создаем пустой словарь вторым способом

d2 = {}

# выводим тип переменной d2

print(type(d2))

Результат:

Создание словаря с данными

# создаем словарь первым способом

d1 = dict(Ivan="manager", Mark="worker")

# выводим словарь 1

print(d1)

# создаем словарь вторым способом

d2 = {"A1":"123", "A2":"456"}

# выводим словарь 2

print(d2)

Результат:

{'Mark': 'worker', 'Ivan': 'manager’}

{'A2': '456', 'A1': '123'}

Добавление элемента в словарь

Чтобы добавить элемент в словарь нужно указать новый ключ и значение.

# создание словаря

d1 = {"Russia":"Moscow", "USA":"Washington"}

# добавление ключа и значения в словарь

d1["China"]="Beijing"

# вывод словаря

print(d1)

Результат:

{'Russia': 'Moscow', 'China': 'Beijing', 'USA': 'Washington'}

Удаление элемента

# создаем словарь

d2 = {"A1":"123", "A2":"456"}

# удаляем ключ и его значение из словаря

del d2["A1"]

# вывод словаря

print(d2)

Результат:

{'A2': '456'}

Проверка наличия ключа в словаре

# создаем словарь

d2 = {"A1":"123", "A2":"456"}

# проверяем есть ли в словаре ключ A1

print("A1" in d2)

# проверяем есть ли в словаре ключ A3

print("A3" in d2)

Результат:

True

False

Доступ к элементу словаря

# создаем словарь

d1 = {"Russia":"Moscow", "USA":"Washington"}

# выводим значение по ключу “Russia”

print(d1["Russia"])

Результат:

'Moscow'

clear() - удаляет все элементы словаря.

# создаем словарь

d2 = {"A1":"123", "A2":"456"}

# выводим словарь

print(d2)

# удаляем все элементы в словаре

d2.clear()

# выводим словарь

print(d2)

Результат:

{'A2': '456', 'A1': '123’}

{}

copy() - создается новая копия словаря.

# создаем словарь

d2 = {"A1":"123", "A2":"456"}

# копируем словарь d2 в d3

d3 = d2.copy()

# выводим словарь d3

print(d3)

# меняем значение по ключу A1

d3["A1"]="789"

# выводим словарь d2

print(d2)

# выводим словарь d3

print(d3)

Результат:

{'A1': '123', 'A2': '456’}

{'A1': '123', 'A2': '456’}

{'A1': '789', 'A2': '456'}

get(key) - возвращает значение из словаря по ключу key.

# создаем словарь

d = {"A1":"123", "A2":"456"}

# выводим значение по ключу А1

print(d.get("A1"))

Результат:

'123'

items() - возвращает элементы словаря (ключ, значение) в отформатированном виде.

# создаем словарь

d = {"A1":"123", "A2":"456"}

# выводим ключи и значения

print(d.items())

Результат:

dict_items([('A1', '123'), ('A2', '456')])

keys() - возвращает ключи словаря.

# создание словаря

d = {"A1":"123", "A2":"456"}

# вывод всех ключей

print(d.keys())

Результат:

dict_keys(['A1', 'A2'])

pop(key[, default])

Если ключ key есть в словаре, то данный элемент удаляется из словаря и возвращается значение по этому ключу, иначе будет возвращено значение default. Если default не указан и запрашиваемый ключ отсутствует в словаре, то будет вызвано исключение KeyError.

# создание словаря

d = {"A1":"123", "A2":"456"}

# удаляем и выводим значение по ключу A1

print(d.pop("A1"))

# выводим словарь

print(d)

Результат:

123

{'A2': '456'}

popitem()

Удаляет и возвращает пару (ключ, значение) из словаря. Если словарь пуст, то будет вызвано исключение KeyError.

# создаем словарь

d = {"A1":"123", "A2":"456"}

# удаляем выводим ключ и значение

print(d.popitem())

# выводим словарь

print(d)

Результаты:

('A2', '456’)

{'A1': '123'}

setdefault(key[, default])

Если ключ key есть в словаре, то возвращается значение по ключу. Если такого ключа нет, то в словарь вставляется элемент с ключом key и значением default, если default не определен, то по умолчанию присваивается None.

# создание словаря

d = {"A1":"123", "A2":"456"}

# так как ключа A3 нет в словаре то добавляем

d.setdefault("A3", "777")

# выводим словарь

print(d)

# выводим значение по ключу A1

print(d.setdefault("A1"))

Результат:

{'A1': '123', 'A2': '456', 'A3': '777'}

123

update([other])

Обновить словарь парами (key/value) из other, если ключи уже существуют, то обновить их значения.

# создаем словарь

d = {"A1":"123", "A2":"456"}

# обновляем значения словаря

d.update({"A1":"333", "A3":"789"})

# выводим словарь

print(d)

Результат:

{'A2': '456', 'A3': '789', 'A1': '333'}

values()

Возвращает значения элементов словаря.

# создаем словарь

d = {"A1":"123", "A2":"456"}

# выводим значения словаря

print(d.values())

Результат:

dict_values(['456', '123'])

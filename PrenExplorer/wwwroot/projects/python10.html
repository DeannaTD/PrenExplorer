Урок 10. Списки

Список (list) – это структура данных для хранения объектов различных типов. 

Создание списка

Например:

# создание пустого списка

a = []

b = list()

# создание списка с заданным набором данных

c = [1, 2, 3]

# вывод типа переменных

print(type(a))

print(type(b))

print(type(c))

Результат:

Если у вас уже есть список и вы хотите создать его копию, то можно воспользоваться следующим из способов:

# создаем список c

c = [1, 2, 3]

# копируем список c в переменную a

a = c[:]

# копируем список c в переменную b

b = list(c)

# выводим все списки

print(a)

print(b)

print(c)

Результат:

[1, 2, 3]

[1, 2, 3]

[1, 2, 3]

Добавление элемента в список

Добавление элемента в список осуществляется с помощью метода append().

# создаем пустой список

a = []

# добавляем в него элемент 3

a.append(3)

# добавляем в него элемент ‘hello’

a.append('hello')

# выводим список

print(a)

Результат:

[3, 'hello']

Удаление элемента из списка

Для удаления элемента из списка, в случае если вы знаете его значение, используйте метод remove(x), при этом будет удалена первая ссылка на данный элемент.

# создаем список

a = [3, 'hello', 3]

# удаляем элемент со значением 3

a.remove(3)

# выводим список

print(a)

Результат:

['hello', 3]

Удаление элемента по индексу

Если необходимо удалить элемент по его индексу, воспользуйтесь командой del имя_списка[индекс].

# создаем список

a = [3, 'hello', 3, 5, 18]

# удаляем элемент под индексом 4

del a[4]

# выводим список

print(a)

Результат:

[3, 'hello', 3, 5]

Изменение значения по индексу

Изменить значение элемента списка, зная его индекс, можно напрямую к нему обратившись.

# создаем список

a = [3, 'hello', 3, 5, 18]

# изменяем значение под индексом 1

a[1] = 4

# выводим список

print(a)

Результат:

[3, 4, 3, 5, 18]

Очистить список

Очистить список можно просто заново его проинициализировав, так как будто вы его вновь создаете.

# создаем список

a = [3, 'hello', 3, 5, 18]

# очищаем список

a = []

# выводим список

print(a)

Результат:

[]

Получение значения

Для получения доступа к элементу списка укажите индекс этого элемента в квадратных скобках.

# создаем список

a = [3, 'hello', 3, 5, 18]

# выводим из списка элемент под индексом 2

print(a[2])

Результат:

3

Можно использовать отрицательные индексы, в таком случае счет будет идти с конца, например для доступа к последнему элементу списка можно использовать вот такую команду:

# выводим из списка последний элемент

print(a[-1])

Результат:

18

Для получения из списка некоторого подсписка в определенном диапазоне индексов, укажите начальный и конечный индекс в квадратных скобках, разделив их двоеточием.

# выводим из списка элементы со второго по четвертый индекс

print(a[2:4])

Результат:

[3, 5]

Расширяет список – extend()

# создаем список a

a = [3, 'hello', 3, 5, 18]

# создаем список b

b = [-5, 16]

# вставляем в список a список b

a.extend(b)

# выводим список a

print(a)

Результат:

[3, 'Привет', 3, 5, 18, -5, 16]

Вставка элемента по индексу

Вставить элемент x в позицию i.  Первый аргумент – индекс элемента, после которого будет вставлен элемент x.

# создаем список

a = [3, 'hello', 3, 5, 18]

# вставляем элемент 5 под индексом 0

a.insert(0, 5)

# выводим список

print(a)

# вставляем элемент 9, в качестве индекса берем длину списка, т.е. вставляем элемент 9 в конец

a.insert(len(a), 9)

# выводим список

print(a)

Результат:

[5, 3, 'Привет', 3, 5, 18]

[5, 3, 'Привет', 3, 5, 18, 9]

Метод pop()

Удаляет элемент из позиции i и возвращает его. Если использовать метод без аргумента, то будет удален последний элемент из списка.

# создаем список

a = [3, 'hello', 3, 5, 18]

# удаляем и выводим последний элемент

print(a.pop())

# удаляем и выводим элемент под индексом 2

print(a.pop(2))

Результат:

18

3

Удаление всех элементов из списка – clear()

# создаем список

a = [3, 'hello', 3, 5, 18]

# очищаем список

a.clear()

# выводим список

print(a)

# создаем список

b = [11, 25, 45]

# очищаем список другим способом

del b[:]

# выводим список

print(b)

Результат:

[]

[]

Возвращает индекс элемента – index()

# создаем список

a = [3, 'hello', 3, 5, 18]

# выводим индекс элемента 5

print(a.index(5))

Результат:

3

Возвращает количество вхождений элемента в список – count()

# создаем список

a = [3, 'hello', 3, 5, 18]

# выводим количество элемента 3 в списке

print(a.count(3))

Результат:

2

Сортировка элементов – sort()

a = [3, 61, 3, 5, 18]

a.sort() # по возрастанию

print(a)

a.sort(reverse=True) # по убыванию

print(a)

Результат:

[3, 3, 5, 18, 61]

[61, 18, 5, 3, 3]

Изменяет порядок расположения элементов в списке на обратный – reverse()

# создаем список

a = [3, 61, 3, 5, 18]

# меняем порядок на обратный

a.reverse()

# выводим список

print(a)

Результат:

[18, 5, 3, 61, 3]

Возвращает копию списка – copy()

# создаем список

a = [3, 61, 3, 5, 18]

# копируем список a в переменную b

b = a.copy()

# выводим список b

print(b)

Результат:

[3, 61, 3, 5, 18]

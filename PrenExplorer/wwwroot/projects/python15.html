Урок 15. Множества.

Множество в python - "контейнер", содержащий не повторяющиеся элементы в случайном порядке.

Создание множества

# создаем пустое множество

a = set()

# создаем множество и заполняем его

b = set('hello’)

# выводим множество

print(b)

# создаем и заполняем множество

num_set = {1, 2, 3, 4, 5, 6} 

# выводим множество

print(num_set)

Результат:

{'h', 'o', 'e', 'l'}

{1, 2, 3, 4, 5, 6}

Получение всех элементов множества

months = set(["Jan", "Feb", "March", "Apr", "May", "June", "July", "Aug", "Sep", "Oct", "Nov", "Dec"])

for m in months:

print(m)

Результат:

Проверка есть ли в данном множестве элемент May

months = set(["Jan", "Feb", "March", "Apr", "May", "June", "July", "Aug", "Sep", "Oct", "Nov", "Dec"])

print("May" in months)

Результат:

Добавление элементов во множество

months = set(["Jan", "March", "Apr", "May", "June", "July", "Aug", "Sep", "Oct", "Nov", "Dec"])

months.add("Feb")

print(months)

Результат:

{'June', 'Sep', 'Feb', 'Jan', 'March', 'July', 'Aug', 'Dec', 'Nov', 'Apr', 'May', 'Oct'}                                

Удаление элемента из множества

num_set = {1, 2, 3, 4, 5, 6}  

num_set.discard(3)  

print(num_set)

Результат:

{1, 2, 4, 5, 6}

метод discard() не будет выдавать ошибку, если элемент не был найден во множестве.

pop()

С методом pop(), мы можем удалить и вернуть элемент. Так как элементы находятся в произвольном порядке, мы не можем утверждать или предсказать, какой элемент будет удален.

num_set = {1, 2, 3, 4, 5, 6}

print(num_set.pop())

Результат:

1

clear()

Метод Python под названием clear() поможет удалить все элементы во множестве.

num_set = {1, 2, 3, 4, 5, 6}

num_set.clear()

print(num_set)

Результат:

set()

Объединение множеств

Первый способ:

months_a = set(["Jan", "Feb", "March", "Apr", "May", "June"])  

months_b = set(["July", "Aug", "Sep", "Oct", "Nov", "Dec"])

all_months = months_a.union(months_b)  

print(all_months)

Результат:

{'Oct', 'Jan', 'Nov', 'May', 'Aug', 'Feb', 'Sep', 'March', 'Apr', 'Dec', 'June', 'July'}

Второй способ:

months_a = set(["Jan","Feb", "March", "Apr", "May", "June"])  

months_b = set(["July", "Aug", "Sep", "Oct", "Nov", "Dec"])

print(months_a | months_b)

Результат:

{'Feb', 'Apr', 'Sep', 'Dec', 'Nov', 'June', 'May', 'Oct', 'Jan', 'July', 'March', 'Aug'}

Пересечение множеств

Есть два множества: А и В. Их пересечение представляет собой множество элементов, которые являются общими для А и для В.

Первый способ:

x = {1, 2, 3}  

y = {4, 3, 6}

print(x & y)

Результат:

3

Второй способ:

x = {1, 2, 3}  

y = {4, 3, 6}

z = x.intersection(y)  

print(z)

Результат:

3

Разница между множествами

Есть два множества: А и В. Разница между А и В (А — В) — это множество со всеми элементами, которые содержатся в А, но не в В. Соответственно, (В — А) — это множество со всеми элементами в В, но не в А.

Первый способ:

set_a = {1, 2, 3, 4, 5}

set_b = {4, 5, 6, 7, 8}

diff_set = set_a.difference(set_b)

print(diff_set)

Результат:

{1, 2, 3}

set_a = {1, 2, 3, 4, 5}

set_b = {4, 5, 6, 7, 8}

print(set_a - set_b)

Результат:

{1, 2, 3}

Симметричная разница

между множествами А и В — это множество с элементами, которые находятся в А и В, за исключением тех элементов, которые являются общими для обеих множеств. Это определяется использованием метода Python под названием symmetric_difference(), или оператора ^.

Первый способ:

set_a = {1, 2, 3, 4, 5}

set_b = {4, 5, 6, 7, 8}

symm_diff = set_a.symmetric_difference(set_b)

print(symm_diff)

Результат:

{1, 2, 3, 6, 7, 8}

Второй способ:

set_a = {1, 2, 3, 4, 5}

set_b = {4, 5, 6, 7, 8}

print(set_a ^ set_b)

Результат:

{1, 2, 3, 6, 7, 8}

Сравнение множеств

a = set([1, 2, 3, 4, 5, 6, 7])

b = set([1, 2, 3, 4])

print(a <= b)

print(b <= a)

print(a.issubset(b))

print(b.issubset(a))

Результат:

False

True

False

True

Метод copy()

Этот метод возвращает копию множества. Например:

string_set = {"Nicholas", "Michelle", "John", "Mercy"}

x = string_set.copy()

print(x)

Результат:

{'Mercy', 'Nicholas', 'Michelle', 'John'}

Метод isdisjoint()

Этот метод проверяет, является ли множество пересечением или нет. Если множества не содержат общих элементов, метод возвращает True, в противном случае — False. Например:

names_a = {"Nicholas", "Michelle", "John", "Mercy"}

names_b = {"Jeff", "Bosco", "Teddy", "Milly"}

x = names_a.isdisjoint(names_b)

print(x)

Результат:

True

Метод len()

Этот метод возвращает длину множества, которая является общим количеством элементов во множестве. Пример:

names_a = {"Nicholas", "Michelle", "John", "Mercy"}

print(len(names_a))

Результат:

4

Frozenset

Frozenset (замороженное множество) – это класс с характеристиками множества, однако, как только элементы становятся назначенными, их нельзя менять. Кортежи могут рассматриваться как неизменяемые списки, в то время как frozenset-ы — как неизменные множества.

X = frozenset([1, 2, 3, 4, 5, 6])

Y = frozenset([4, 5, 6, 7, 8, 9])

print(X)

print(Y)

Результат:

frozenset({1, 2, 3, 4, 5, 6})

frozenset({4, 5, 6, 7, 8, 9})

Входные данные

На вход программе подаётся символьная строка.

Выходные данные

Программа должна вывести в одной строке все различные цифры, которые встречаются в исходной строке, в порядке возрастания. Если в строке нет цифр, нужно вывести слово 'NO'.

входные данные

ab1n32kz2

выходные данные

123

входные данные

asdasd

выходные данные

NO

Входные данные

Выходные данные

Программа должна вывести в одну строчку в порядке возрастания все цифры, встречающиеся во входной строке больше одного раза. Если таких цифр нет, нужно вывести слово 'NO'.

2

Выходные данные

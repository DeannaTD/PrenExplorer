Урок 4. Строки. Базовые операции

Конкатенация (сложение) строк.

+ — оператор конкатенации строк. Он возвращает строку, состоящую из других строк, как показано здесь

S1 = 'spam’

S2 = 'eggs’

print(S1 + S2)

Результат

'spameggs'

Дублирование строки

* — оператор создает несколько копий строки.

print('spam' * 3)

Результат:

spamspamspam

Длина строки (функция len)

len('spam’)

Результат:

4

Доступ по индексу

Отчет индекса начинается не с 1 а с 0.

S = 'spam’

print(S[0])

print(S[2])

print(S[-2])

Результат:

s

a

a

Как видно из примера, в Python возможен и доступ по отрицательному индексу, при этом отсчет идет от конца строки.

Извлечение среза

Оператор извлечения среза: [X:Y]. X – начало среза, а Y – окончание;

символ с номером Y в срез не входит. По умолчанию первый индекс равен 0, а второй - длине строки.

s = 'spameggs'

print(s[3:5])

print(s[2:-2])

print(s[:6])

print(s[1:])

print(s[:])

Результат:

me

ameg

spameg

pameggs

spameggs

Извлечение среза с шагом

Существует еще один вариант синтаксиса среза, о котором стоит упомянуть. Добавление дополнительного : и третьего индекса означает шаг, который указывает, сколько символов следует пропустить после извлечения каждого символа в срезе.

s = 'spameggs'

print(s[::-1])

print(s[3:5:1])

print(s[2::2])

Результат:

sggemaps

me

aeg

Замена шаблона

s = 'spameggs'

s = s.replace('eggs','old')

print(s)

Результат:

spamold

Заменяет ‘eggs’ на ‘old’

Поиск подстроки в строке

s = 'spameggs'

s = s.index('meg')

print(s)

Результат:

3

Находит с какого индекса начинается слово ‘meg’

Разбиение строки по разделителю

s = 'spameggs'

s = s.split('meg')

print(s)

Результат:

['spa', 'gs']

Дополнительные операции:

S.isdigit() - Состоит ли строка из цифр

S.isalpha() - Состоит ли строка из букв

S.isalnum() - Состоит ли строка из цифр или букв

S.islower() - Состоит ли строка из символов в нижнем регистре

S.isupper() - Состоит ли строка из символов в верхнем регистре

S.istitle() - Начинаются ли слова в строке с заглавной буквы

S.upper() - Преобразование строки к верхнему регистру

S.lower() - Преобразование строки к нижнему регистру

S.startswith(str) - Начинается ли строка S с шаблона str

S.endswith(str) - Заканчивается ли строка S шаблоном str

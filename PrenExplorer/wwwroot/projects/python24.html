Урок 24. Приватные атрибуты.

Атрибуты класса -- переменные, методы (пока не будем рассматривать раздельно атрибуты класса и атрибуты экземпляра класса). 

Приватные атрибуты предназначены только для внутреннего использования. Их нельзя вызывать/использовать извне. По факту, это можно сделать, только не рекомендуется.

Приватные переменные -- это те же переменные, только пишутся c нижним подчеркиванием в начале: _private_variable. 

Приватный метод -- служебный метод -- деталь имплементации, и его поведение может быть изменено без предупреждения. Также пишется через нижнее подчеркивание:

_private_method()

Задания:

1. Создайте класс Student. При создании его экземпляра, мы должны записать имя и фамилию студента в соответствующие переменные объекта. В классе должны быть:

a. переменная объекта books = [ ]

b. переменная объекта “knowledge” со значением по умолчанию 0

c. метод read_book, который принимает название книги, добавляет книгу в books, добавляет 100 баллов в knowledge

d. метод do_homework, который при вызове добавляет 10 баллов в knowledge

e. Создайте экземпляры, вызовите методы.

2. Добавьте в класс Student переменную объекта is_ready_to_work = False.

a) Добавьте метод do_real_world_project, при вызове которого в knowledge добавляется 50 баллов.

b) Добавьте приватный метод _add_points, который принимает баллы, добавляет в knowledge. Примените этот метод вместо кода с аналогичным функционалом в методах read_book и do_homework.

c) Добавьте следующую проверку в метод _add_points: если значение knowledge больше 2000, то выводим текст “This student is ready to work” и записывает True в is_ready_to_work.

d) Создайте экземпляры, вызовите все методы.

3. В класс Student добавьте переменную экземпляра languages = { }. Добавьте метод learn_new_language, который принимает название языка программирования, и целочисленное от 1 до 100, добавляет в словарь languages язык в качестве ключа, баллы в качестве значения. Выкидывает ошибку ValueError, если балл меньше 1 и больше 100.

4. Создайте класс BankAccount, у которого должны быть:

a. переменные объекта

i. balance (со значением 0 по умолчанию)

ii. account_id (со значением random.randrange(100000,199999))

iii. transactions_history (c пустым словарем)

b. методы

I. withdraw (снять) принимает сумму, которую нужно снять с баланса и возвращает остаток баланса после снятия

II. deposit (положить) принимает сумму, которую нужно положить на баланс и возвращает остаток баланса после пополнения

III. После создания методов и переменных, попробуйте создать счет, пополнить баланс и снять деньги с баланса.

5. В созданный класс BankAccount добавьте методы:

a) receive (принять) который принимает сумму, увеличивает свой баланс, записывает счет отправителя в словарь transactions_history в качестве ключа, и сумму добавляет в список, который является значением. Почему список? Потому что переводов от данного банковского счета может быть много (пока дату перевода не будем записывать).

b) transfer (перевести), который принимает сумму и другой экземпляр класса BankAccount (параметр назовём receiving_account), на который нужно перевести деньги. В результате работы этого метода нужно уменьшить сумму на балансе, вызвать метод receive у receiving_account. Метод должен возвращать остаток денег на балансе после перевода.

c) После реализации этих методов, попробуйте создать несколько счетов, попробуйте совершить несколько денежных переводов.

6. Создайте класс Student, конструктор которого имеет параметры name, lastname, department, year_of_entrance. Добавьте метод get_student_info, который возвращает имя, фамилию, год поступления и факультет в отформатированном виде: “Вася Пупкин, поступил в 2017 г. Факультет: Дата Саентологии.”

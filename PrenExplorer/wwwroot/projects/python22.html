Урок 22. Условный оператор в генераторах списков

УСЛОВНЫЙ ОПЕРАТОР IF В ГЕНЕРАТОРАХ СПИСКОВ

Условный оператор if в генераторе списка работает почти также, как в случаях, когда они работают как отдельная конструкция. В генераторах условие пишется в конце, после перечисления, которое мы перебираем. Например, в следующем генераторе списка условие выделено красным:

cubelist=[x**3 for x in range(1,11) if x%2 != 0]

print(cubelist)

Разберем более сложный пример, в котором в конце мы получаем множество, состоящее из согласных.

vowels = ('A', 'E', 'I', 'O', 'U', 'a', 'e', 'i', 'o', 'u')

a_str = 'This is a test string for list comprehension showcase'

unique_consonants = set([x for x in a_str if x not in vowels and len(x.strip())])

print(unique_consonants)

Для улучшения читабельности, это можно оформить следующим образом (кстати, такое оформление часто встречается в реальной жизни; это выполнимый код):

vowels = ('A', 'E', 'I', 'O', 'U', 'a', 'e', 'i', 'o', 'u')

a_str = 'This is a test string for list comprehension showcase'

unique_consonants = set(

     [

          x for x in a_str

          if x not in vowels and len(x.strip())

     ]

)

print(unique_consonants)

Во фрагменте x for x in a_str происходит перебор букв строки a_str, а в куске кода if x not in vowels происходит проверка того, не содержится ли буква в кортеже vowels. А len(x.strip()) проверяет, не пустое ли это строковое значение, если убрать в нем пробелы.

Фрагмент x for x in a_str срабатывает только если эти две проверки прошли успешно.
